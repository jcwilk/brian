## Features

- **Full-text search** across all your knowledge
- **Tag-based organization** with autocomplete
- **Markdown support** for rich formatting
- **Code syntax highlighting** for snippets
- **Link metadata extraction** for previews
- **Knowledge graph** showing connections
- **Desktop app** with quick capture hotkey
- **Export/Import** for portability

## Tech Stack

- **Backend**: Python + FastAPI
- **Database**: SQLite (inspired by Goose's architecture)
- **Frontend**: Modern JavaScript + HTML/CSS
- **Desktop**: Electron/Tauri wrapper
- **Visualization**: D3.js for graphs and timelines
